# dodo

dodo 是一个使用 HTML+JS+CSS 编写的待办事项软件，基于 jQuery+jQTouch，通过使用 [PhoneGap](http://phonegap.com/) 封装，可以运行于 iOS、webOS、Android 等各个平台。

## 功能

* 添加、编辑、删除待办事项
* 列表按完成日期排列
* 支持重要事项标记
* 设置待办事项默认完成时间
* 清除已完成待办事项
